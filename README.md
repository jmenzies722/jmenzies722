### What I Ship

**[AWS Architect MCP](https://github.com/jmenzies722/aws-architect-mcp)**  
MCP server that turns an AI agent into an AWS solutions architect. Give it a goal — it selects a reference pattern, designs the architecture, and emits the Terraform, IAM policy, cost estimate, and scaffolded project layout.

**[Lore](https://github.com/jmenzies722/lore)**  
Agent governance layer. Org context, permissions, and audit trail delivered as an MCP server. One config block gives any AI agent full organizational awareness.

**[Forge](https://github.com/jmenzies722/forge)**  
AWS-native internal developer platform. Golden paths, self-service deployments, and AI agents on ECS Fargate.

**[ARIA Skills Hub](https://github.com/jmenzies722/aria-skills-hub)**  
Custom Claude Code skills for engineers — HOMELAB OS and more. Install in one line.

**[claude-setup](https://github.com/jmenzies722/claude-setup)**  
Claude Code environment auditor. Scores your setup 0–100, detects your profile, generates CLAUDE.md, fixes atomically.
