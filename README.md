### What I Ship

**[Axiom](https://github.com/jmenzies722/axiom)** `private`  
npm + Docker Hub for AI agents. Publish, install, and run agents across any protocol. The `axiom.yaml` manifest is the Dockerfile of the agent era.

**[Lore](https://github.com/jmenzies722/lore)**  
Agent governance layer. Org context, permissions, and audit trail delivered as an MCP server. One config block gives any AI agent full organizational awareness.

**[Forge](https://github.com/jmenzies722/forge)**  
AWS-native internal developer platform. Golden paths, self-service deployments, and AI agents on ECS Fargate.

**[Luminatra](https://github.com/jmenzies722/luminatra)** `private`  
Developer automation platform. Create AI agents, chain them into workflows, execute with full serverless history.

**[Auris](https://github.com/jmenzies722/auris)** `private`
